Emextal
====

_The wiser cheese_

Local ML voice chat using high-end models, aiming for a self contained, user-friendly out-of-the-box experience as much as possible.

This is an experimental variant of [Emeltal](https://github.com/ptsochantaris/emeltal) with some important differences:
- Based on the Apple's MLX framework.
- Less exploration of experimental models, focusing on well known and reliable releases.
- Specific focus on the conversational experience; technical coding and agentic work is done far better with other apps like LM/Unsloth Studio, Ollama, and so on.
